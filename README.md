#  App de métricas con SwiftUI
=========================================================================

📱💻 Esta aplicación desarrollada con el framework SwiftUI, permitirá tomar
medidas del uso de la CPU, memoria y energía en dispositivos iOS.

 El propósito del desarrollo del app es de carácter académico-investigativo,
 ya que fue diseñada para mi tesis de grado. Con esta app se obtendrá el 
 grado de impacto que se tiene con la calidad de software, especialmente
 con la dimensión del Rendimiento.
 
 ## Guía de ejecución
 ------------------------------------------------------------------------
 Simplemente se tiene que clonar el proyecto y ejecutarlo en Xcode. Para
 obtener las métricas se debe de usar Xcode Instruments y ejecutar el
 app en dispositivos iPhone de manera física para obtener resultados más
 confiables.
